SCRIPTINA FONT — for the "Nolwazi Mkhabela" name logo
======================================================

The site's name logo is styled with the font "Scriptina" by Apostrophic
Labs. Font files can't be bundled automatically, so add it yourself:

1. Download "Scriptina" from 1001fonts:
   https://www.1001fonts.com/scriptina-font.html

2. Unzip it. You'll typically get a .ttf or .otf file.

3. Rename the file to "Scriptina" and place it in THIS folder, e.g.:
      fonts/Scriptina.ttf     (or .otf)

   The site already looks for any of these, in order:
      fonts/Scriptina.woff2
      fonts/Scriptina.woff
      fonts/Scriptina.ttf
      fonts/Scriptina.otf

   (Best performance: convert the .ttf/.otf to .woff2 with a free
    tool like transfonter.org, then drop Scriptina.woff2 here.)

Until the file is added, the logo gracefully falls back to the
"Dancing Script" signature font so nothing looks broken.

LICENCE: Scriptina is free for personal use. Check the licence on the
1001fonts page before using it on a commercial/live site.
